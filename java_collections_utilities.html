Collections Utility Methods

The Collections class provides static methods for sorting, searching, reversing, shuffling, and synchronizing collections.

  • sort(): Sorts a list
  • reverse(): Reverses a list
  • shuffle(): Randomizes order
  • binarySearch(): Searches a sorted list
  • synchronizedList(): Returns a thread-safe list
  • unmodifiableList(): Returns a read-only list

Example

import java.util.*;

List<Integer> list = Arrays.asList(3, 1, 2);
Collections.sort(list); // [1, 2, 3]
Collections.reverse(list); // [3, 2, 1]
Collections.shuffle(list); // Random order
int idx = Collections.binarySearch(list, 2); // index of 2 (if sorted)
List<Integer> syncList = Collections.synchronizedList(new ArrayList<>());

Tip: Use Collections utility methods for common collection operations.
