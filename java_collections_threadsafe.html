Thread-safe Collections

Java provides several ways to create thread-safe collections:

  • Legacy synchronized classes: Vector, Hashtable
  • Collections utility methods: Collections.synchronizedList(), etc.
  • Concurrent collections: ConcurrentHashMap, CopyOnWriteArrayList, BlockingQueue
import java.util.concurrent.*;

ConcurrentHashMap<String, Integer> cmap = new ConcurrentHashMap<>();
cmap.put("A", 1);

Prefer concurrent collections for high-concurrency scenarios.
