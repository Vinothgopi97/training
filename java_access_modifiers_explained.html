Java Access Modifiers Explained

Access modifiers in Java determine the visibility (scope) of classes, methods, and variables. There are four main access modifiers:

Modifier Accessible within same class Accessible within same package Accessible in subclass (other package) Accessible everywhere
private Yes No No No
default (no modifier) Yes Yes No No
protected Yes Yes Yes No
public Yes Yes Yes Yes

Where Can Access Modifiers Be Used?

Access modifiers can be applied to the following elements in Java:

Element public protected default
(no modifier)		 private
Top-level class ✔ ✔
Inner class ✔ ✔ ✔ ✔
Methods ✔ ✔ ✔ ✔
Variables (fields) ✔ ✔ ✔ ✔
Constructors ✔ ✔ ✔ ✔
  • Top-level classes can only be public or default (no modifier).
  • Inner classes, methods, fields, and constructors can use any access modifier.
Examples: 
// Top-level class (public or default only)
public class MyClass { ... }
class DefaultClass { ... }

// Inner class with private access
class Outer {
    private class Inner { ... }
}

// Method with protected access
class Example {
    protected void show() { ... }
}

// Field with private access
class Person {
    private String name;
}

1. private

The private modifier makes a member accessible only within its own class. 

// Example: Private variable
class Person {
    private String name;

    public void setName(String n) {
        name = n;
    }
    public String getName() {
        return name;
    }
}

class Test {
    public static void main(String[] args) {
        Person p = new Person();
        // p.name = "Alice"; // Error: name has private access
        p.setName("Alice");
        System.out.println(p.getName()); // Alice
    }
}

2. default (no modifier)

If no access modifier is specified, it is default (also called package-private). The member is accessible only within classes in the same package. 

// Example: Default access
class Animal {
    int age; // default access
}

class Test {
    public static void main(String[] args) {
        Animal a = new Animal();
        a.age = 5; // OK if Test is in the same package as Animal
    }
}

3. protected

The protected modifier allows access within the same package and in subclasses (even if they are in different packages). 

// Example: Protected variable
class Vehicle {
    protected String type = "Car";
}

class Car extends Vehicle {
    public void printType() {
        System.out.println(type); // Accessible in subclass
    }
}

4. public

The public modifier allows the member to be accessed from any other class. 

// Example: Public method
public class HelloWorld {
    public void sayHello() {
        System.out.println("Hello, world!");
    }
}

class Test {
    public static void main(String[] args) {
        HelloWorld hw = new HelloWorld();
        hw.sayHello(); // Accessible everywhere
    }
}
