Mini Project: Console-Based Mini Banking System

Project Overview:
Build a console-based Mini Banking System that allows users to:

  • Create accounts (Savings/Current)
  • Deposit and withdraw money
  • View account details and transaction history
  • Transfer funds between accounts
  • Generate account statements

Requirements & Concepts to Cover:

  • OOP: Classes for Account, Customer, Transaction, Bank, etc.; inheritance for account types; encapsulation for data hiding.
  • Collections: Store accounts, customers, and transactions using List, Map, etc.
  • Exception Handling: Handle invalid operations (e.g., insufficient funds, invalid account).
  • Java 8 Features: Use streams for searching/filtering, lambda expressions for sorting, functional interfaces, Optional for safe retrievals.
  • Date/Time API: Record transaction timestamps.
  • File I/O (optional): Save/load data for persistence.
  • Interfaces & Abstract Classes: For account operations and transaction processing.
  • Default/Static Methods in Interfaces: For utility methods.
  • Method References: For concise code in stream operations.

Sample Use Cases:

  • Create a new customer and open an account
  • Deposit/withdraw money
  • Transfer money between accounts
  • Print transaction history for an account (using streams and lambdas)
  • Handle errors gracefully (e.g., invalid account, insufficient funds)

Bonus: Add features like password protection, interest calculation, or file-based persistence.
