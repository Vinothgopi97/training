PriorityQueue in Java

PriorityQueue is a queue that orders elements according to their natural ordering or by a Comparator.

  • Does not allow null elements
  • Not thread-safe
  • Elements are ordered, not necessarily FIFO

Example

import java.util.*;

PriorityQueue<Integer> pq = new PriorityQueue<>();
pq.add(10);
pq.add(5);
pq.add(20);
System.out.println(pq); // [5, 10, 20] (order may vary)
System.out.println(pq.poll()); // 5 (smallest element)
System.out.println(pq); // [10, 20]

Common PriorityQueue Methods

  • add(E e): Inserts element.
  • offer(E e): Inserts element, returns false if full.
  • poll(): Retrieves and removes head (smallest/largest), returns null if empty.
  • peek(): Retrieves head without removing, returns null if empty.
  • remove(Object o): Removes specified element.
  • size(): Number of elements.
  • clear(): Removes all elements.
  • isEmpty(): Checks if queue is empty.
  • iterator(): Returns an iterator over the elements.

Tip: Use PriorityQueue for scheduling tasks by priority.
