Java OOPs Assignment

Implement Java programs for the following questions to practice and demonstrate your understanding of Object Oriented Programming concepts.

  1. Class and Object:
    Create a Student class with fields name and rollNo. Write a program to create and display the details of two students.
    Example scenario: Input student details from the user and print them.
  2. Constructor Overloading:
    Create a Rectangle class with length and width. Provide a default constructor (sets both to 1) and a parameterized constructor. Show the area for both cases.
    Example scenario: Demonstrate both constructors in main and print area.
  3. Single Inheritance:
    Create a Person class with name and age. Create an Employee class that inherits from Person and adds employeeId and salary. Display employee details.
    Example scenario: Create an employee object and print all details.
  4. Multilevel Inheritance:
    Create classes Animal (method: eat()), Mammal (method: walk()), and Dog (method: bark()). Show method calls from all levels.
    Example scenario: Create a Dog object and call all three methods.
  5. Hierarchical Inheritance:
    Create a base class Vehicle (method: start()) and two derived classes Car and Bike (override start()).
    Example scenario: Create objects of both Car and Bike and call start().
  6. Method Overloading:
    Write a Calculator class with overloaded add methods: add(int, int), add(double, double), add(int, int, int).
    Example scenario: Call all overloaded methods and print results.
  7. Method Overriding and Dynamic Polymorphism:
    Create a Shape class with a draw() method. Extend it with Circle and Square classes, each overriding draw(). Demonstrate dynamic method dispatch.
    Example scenario: Use a Shape reference to refer to Circle and Square objects and call draw().
  8. Encapsulation:
    Create a BankAccount class with private fields accountNumber and balance, and public getter/setter methods. Show how to set and get values.
    Example scenario: Set account details and print them.
  9. Abstraction:
    Create an abstract class Appliance with an abstract method turnOn(). Extend it with Fan and Light classes and implement the method.
    Example scenario: Create objects of Fan and Light and call turnOn().
  10. Interface and Multiple Inheritance:
    Create interfaces Printable (method: print()) and Showable (method: show()). Implement both in a class Document and demonstrate calling both methods.
    Example scenario: Create a Document object and call both print() and show().
  11. Use of this and super keywords:
    Write a program with a parent class Base and a child class Derived. Use this to resolve variable shadowing and super to call the parent class constructor and method.
    Example scenario: Show output of both this and super usages.
  12. Static and Instance Initialization Blocks:
    Create a class with a static block, an instance initialization block, and a constructor. Print a message from each to show the order of execution.
    Example scenario: Create two objects and observe the output order.
  13. Object Class Methods:
    Override toString() and equals() in a Book class. Create two Book objects with the same data and show the output of toString() and equals().
    Example scenario: Print both objects and compare them.

Instructions: Write each program in a separate Java file. Add comments to explain your code. Test your programs with sample input and output.
