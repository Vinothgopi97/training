Deque Interface in Java

The Deque (double-ended queue) interface allows elements to be added or removed from both ends. It can be used as a stack (LIFO) or queue (FIFO).

  • Common implementations: LinkedList, ArrayDeque
  • Methods: addFirst, addLast, removeFirst, removeLast, peekFirst, peekLast

Common Deque Methods

  • addFirst(E e): Inserts element at front.
  • addLast(E e): Inserts element at end.
  • removeFirst(): Removes and returns first element.
  • removeLast(): Removes and returns last element.
  • peekFirst(): Retrieves first element without removing.
  • peekLast(): Retrieves last element without removing.
  • offerFirst(E e): Inserts at front, returns false if full.
  • offerLast(E e): Inserts at end, returns false if full.
  • pollFirst(): Retrieves and removes first element, or null if empty.
  • pollLast(): Retrieves and removes last element, or null if empty.
  • size(): Number of elements.
  • isEmpty(): Checks if deque is empty.

Example

import java.util.*;

Deque<String> deque = new ArrayDeque<>();
deque.addFirst("A");
deque.addLast("B");
deque.addFirst("C");
System.out.println(deque); // [C, A, B]
System.out.println(deque.removeLast()); // B
System.out.println(deque); // [C, A]

Tip: Use Deque for both stack and queue operations.
