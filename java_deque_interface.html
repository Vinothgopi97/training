Deque Interface in Java

The Deque (double-ended queue) interface allows elements to be added or removed from both ends. 

Deque<String> deque = new LinkedList<>();
deque.addFirst("A");
deque.addLast("B");
System.out.println(deque); // [A, B]
  • Common implementations: LinkedList, ArrayDeque
  • Can be used as stack or queue
