Method References in Java 8

Method references provide a way to refer to methods of existing classes or objects using a compact syntax. They are often used with lambda expressions and functional interfaces.

Basic Example:


List<String> names = Arrays.asList("A", "B", "C");
names.forEach(System.out::println);

More Examples


// Reference to a static method
Function<String, Integer> strLength = String::length;
System.out.println(strLength.apply("Hello"));

// Reference to an instance method of a particular object
String prefix = "Hello, ";
Function<String, String> addPrefix = prefix::concat;
System.out.println(addPrefix.apply("World"));

// Reference to an instance method of an arbitrary object of a particular type
List<String> words = Arrays.asList("java", "stream", "method");
words.stream().map(String::toUpperCase).forEach(System.out::println);

// Reference to a constructor
Supplier<ArrayList<String>> listSupplier = ArrayList::new;
ArrayList<String> newList = listSupplier.get();

// Reference to array constructor
Function<Integer, String[]> arrayCreator = String[]::new;
String[] arr = arrayCreator.apply(5);
System.out.println("Array length: " + arr.length);

// Sorting with method reference
List<String> fruits = Arrays.asList("Banana", "Apple", "Mango");
fruits.sort(String::compareToIgnoreCase);
System.out.println(fruits);
  • Static method: ClassName::staticMethodName
  • Instance method of object: instance::instanceMethodName
  • Instance method of arbitrary object: ClassName::instanceMethodName
  • Constructor: ClassName::new
