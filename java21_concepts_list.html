Java 21 - Important Concepts (LTS)

1. Virtual Threads (Standard)

Virtual threads are now a standard feature, enabling lightweight concurrency at scale. 


Thread.startVirtualThread(() -> System.out.println("Hello from virtual thread!"));

2. String Templates (Preview)

String templates allow embedding expressions inside string literals for easier string formatting. 


// Example syntax (subject to change as it's a preview feature)
String name = "Java";
String message = STR."Hello, \{name}!";
System.out.println(message); // Hello, Java!

3. Sequenced Collections

New interfaces SequencedCollection, SequencedSet, and SequencedMap provide a uniform API for collections with a defined encounter order. 


SequencedCollection sc = new ArrayList<>();
sc.addFirst("first");
sc.addLast("last");
