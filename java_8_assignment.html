Java 8 Programming Assignments

  1. Write a program to print the current date and time using the Java 8 Date Time API.
  2. Format a given LocalDateTime to the pattern "dd-MM-yyyy HH:mm" and parse a string of this format back to LocalDateTime.
  3. Given two dates, calculate the number of months and days between them using Period.
  4. Check if a given year is a leap year using the Date Time API.
  5. Convert a legacy Date object to Instant and then to LocalDateTime.
  6. Write a method that takes a String and returns an Optional<Integer> if the string can be parsed as an integer, otherwise returns Optional.empty().
  7. Given an Optional<String>, print its value if present, otherwise print "Value not present".
  8. Write a function that takes a list of Optional<String> and returns a list of non-empty strings.
  9. Define a functional interface called Calculator with a method int calculate(int a, int b). Implement addition and multiplication using lambda expressions.
  10. Write a lambda expression to sort a list of strings in reverse alphabetical order.
  11. Use a method reference to print all elements of a list of integers.
  12. Write a program to filter a list of LocalDate objects and collect only the dates that are after today using streams and lambdas.
  13. Use Predicate to check if a string starts with "Java" and use it to filter a list of strings.
  14. Write a method that takes a list of integers and returns their squares using Function and streams.
  15. Demonstrate the use of Supplier to generate random numbers.
  16. Write a program to find the next Monday from today using TemporalAdjusters.
  17. Given a YearMonth, print the number of days in that month.
  18. Write a program to calculate age in years, months, and days from a given birthdate using Period.
  19. Convert a LocalDateTime to ZonedDateTime for a specific time zone and print it.
  20. Write a method that takes a list of strings and prints each string's length using a method reference.
  21. Chain multiple Optional operations to parse a string to integer, double it, and print the result if present.
  22. Write a program to get the first and last day of the current month using the Date Time API.
  23. Demonstrate the use of flatMap with Optional to avoid nested optionals.
  24. Write a program to add and subtract days, months, and years from a given date using ChronoUnit.
  25. List all available zone IDs and print the total count.
