Runnable vs Callable in Java

Feature Runnable Callable
Package java.lang java.util.concurrent
Method void run() V call() throws Exception
Return Value No Yes
Throws Checked Exception No Yes
Used With Thread, ExecutorService ExecutorService, Future

Example: Runnable

class MyRunnable implements Runnable {
    public void run() {
        System.out.println("Runnable running");
    }
}

Example: Callable

import java.util.concurrent.*;

class MyCallable implements Callable<String> {
    public String call() {
        return "Callable result";
    }
}

Summary: Use Runnable for simple tasks that do not return a result. Use Callable when you need a result or need to throw checked exceptions from your task.
