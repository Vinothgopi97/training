Multiple Catch Blocks in Java

Java allows multiple catch blocks after a single try block to handle different types of exceptions separately.

Example

public class MultipleCatchExample {
    public static void main(String[] args) {
        try {
            int[] arr = new int[3];
            arr[5] = 10; // ArrayIndexOutOfBoundsException
            int a = 10 / 0; // ArithmeticException
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.println("Array index error: " + e);
        } catch (ArithmeticException e) {
            System.out.println("Arithmetic error: " + e);
        } catch (Exception e) {
            System.out.println("General exception: " + e);
        }
    }
}

Multi-catch (Java 7+)

You can catch multiple exception types in a single catch block using the pipe (|) symbol. 

try {
    // code
} catch (IOException | SQLException ex) {
    System.out.println("Exception: " + ex);
}
  • Order of catch blocks matters: catch more specific exceptions before general ones.
  • Only one catch block is executed per exception occurrence.
