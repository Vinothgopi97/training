Java OOP: Inheritance

Inheritance allows a class to acquire properties and methods of another class. The class that inherits is called the subclass, and the class being inherited from is the superclass. 

class Animal {
    void eat() {
        System.out.println("Animal eats food");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.eat();  // Inherited method
        d.bark(); // Own method
    }
}
// Output:
// Animal eats food
// Dog barks

Dog inherits from Animal and can use both eat() and bark() methods.
