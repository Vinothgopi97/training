Java 22 - Important Concepts

1. Statements before super()

Java 22 allows statements to appear before super() or this() in constructors, enabling more flexible initialization. 


class Parent {
    Parent(String msg) { System.out.println(msg); }
}
class Child extends Parent {
    Child() {
        String greeting = "Hello from Child";
        super(greeting);
    }
}

2. Unnamed Variables and Patterns (Preview)

The underscore _ can be used as an unnamed variable or pattern, indicating that the value is intentionally ignored. 


if (obj instanceof String _) {
    System.out.println("It's a string, but we don't care about the value.");
}

3. Stream Gatherers (Preview)

Stream gatherers provide a new way to collect and transform data in streams, enabling more advanced data processing patterns. 


// Example usage (API subject to change)
Stream numbers = Stream.of(1, 2, 3, 4, 5);
// Gatherers can be used for advanced grouping, windowing, etc.
