Optional Class in Java 8

The Optional class is a container object which may or may not contain a value. It helps avoid null pointer exceptions and provides methods to deal with optional values.

Basic Example:


Optional<String> name = Optional.ofNullable(null);
System.out.println(name.orElse("Default Name"));

More Examples


// Creating Optional
Optional<String> opt1 = Optional.of("Hello");
Optional<String> opt2 = Optional.empty();

// isPresent and ifPresent
if (opt1.isPresent()) {
    System.out.println(opt1.get());
}
opt1.ifPresent(val -> System.out.println("Value: " + val));

// orElse and orElseGet
String result = opt2.orElse("Fallback");
String result2 = opt2.orElseGet(() -> "Generated Fallback");

// orElseThrow
// Throws exception if value is absent
// opt2.orElseThrow(() -> new RuntimeException("No value present"));

// filter
Optional<String> filtered = opt1.filter(s -> s.startsWith("H"));

// map
Optional<Integer> length = opt1.map(String::length);

// flatMap
Optional<String> upper = opt1.flatMap(s -> Optional.of(s.toUpperCase()));

System.out.println(filtered.orElse("No match"));
System.out.println(length.orElse(-1));
System.out.println(upper.orElse(""));
