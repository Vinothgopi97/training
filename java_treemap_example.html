TreeMap in Java

TreeMap is a Map implementation that stores key-value pairs in sorted order of keys.

  • Does not allow null keys
  • Implements SortedMap and NavigableMap
  • Logarithmic time for basic operations

Example

import java.util.*;

TreeMap<String, Integer> map = new TreeMap<>();
map.put("C", 30);
map.put("A", 10);
map.put("B", 20);
System.out.println(map); // {A=10, B=20, C=30}
System.out.println(map.firstKey()); // A
System.out.println(map.lastKey());  // C

Common TreeMap Methods

  • put(K key, V value): Adds or updates key-value pair.
  • get(Object key): Gets value for key.
  • remove(Object key): Removes mapping for key.
  • containsKey(Object key): Checks if key exists.
  • containsValue(Object value): Checks if value exists.
  • firstKey(): Returns lowest key.
  • lastKey(): Returns highest key.
  • higherKey(K key): Returns least key greater than given key.
  • lowerKey(K key): Returns greatest key less than given key.
  • size(): Number of entries.
  • clear(): Removes all entries.
  • isEmpty(): Checks if map is empty.
  • keySet(): Set of keys.
  • values(): Collection of values.
  • entrySet(): Set of key-value pairs.

Tip: Use TreeMap for sorted key-value storage.
