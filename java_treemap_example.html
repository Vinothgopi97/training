TreeMap Example

TreeMap stores key-value pairs in sorted order of keys. 

import java.util.TreeMap;

TreeMap<String, Integer> map = new TreeMap<>();
map.put("C", 30);
map.put("A", 10);
map.put("B", 20);
System.out.println(map); // {A=10, B=20, C=30}
  • Does not allow null keys
  • Implements SortedMap
