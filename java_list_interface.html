List Interface in Java

The List interface extends Collection and represents an ordered collection (sequence) that may contain duplicate elements. Elements can be accessed by their integer index. 

List<String> list = new ArrayList<>();
list.add("A");
list.add("B");
System.out.println(list.get(0)); // Output: A
  • Common implementations: ArrayList, LinkedList, Vector, Stack
  • Allows duplicates and null values
  • Maintains insertion order
