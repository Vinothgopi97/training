LinkedList Example

LinkedList implements List and Deque interfaces, providing a doubly-linked list. 

import java.util.LinkedList;

LinkedList<String> names = new LinkedList<>();
names.add("Alice");
names.add("Bob");
names.addFirst("Zara");
System.out.println(names); // [Zara, Alice, Bob]
  • Efficient insertions/deletions at both ends
  • Can be used as a queue, stack, or list
