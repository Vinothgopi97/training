LinkedList in Java

LinkedList implements both List and Deque interfaces. It is a doubly-linked list, allowing fast insertions and deletions at both ends.

  • Allows duplicates and nulls
  • Maintains insertion order
  • Can be used as a queue, stack, or list
  • Slower random access than ArrayList

Example

import java.util.*;

LinkedList<String> names = new LinkedList<>();
names.add("Alice");
names.add("Bob");
names.addFirst("Zara");
names.addLast("Charlie");
System.out.println(names); // [Zara, Alice, Bob, Charlie]
names.removeFirst();
System.out.println(names); // [Alice, Bob, Charlie]

Tip: Use LinkedList when you need frequent insertions/removals at the beginning or end.

Common LinkedList Methods

  • addFirst(E e): Inserts element at the beginning.
  • addLast(E e): Appends element at the end.
  • removeFirst(): Removes and returns first element.
  • removeLast(): Removes and returns last element.
  • getFirst(): Returns first element.
  • getLast(): Returns last element.
  • offer(E e): Adds element at the end (queue style).
  • poll(): Retrieves and removes head of the list.
  • peek(): Retrieves head without removing.
