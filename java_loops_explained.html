Java Looping Statements

Java provides several looping constructs to execute a block of code repeatedly:

  • for loop
  • while loop
  • do-while loop
  • for-each loop

for Loop

The for loop is used when the number of iterations is known.

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

for-each Loop

The for-each loop (enhanced for loop) is used to iterate over arrays or collections.

int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}

while Loop

The while loop is used when the number of iterations is not known in advance.

int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

do-while Loop

The do-while loop executes the block at least once before checking the condition.

int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);

Loop Control Statements

  • break: Exits the loop immediately.
  • continue: Skips the current iteration and continues with the next.
  • return: Exits the current method and optionally returns a value.

Example: break

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // exits the loop when i is 5
    }
    System.out.println(i);
}

Example: continue

for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue; // skips the rest of the loop when i is 2
    }
    System.out.println(i);
}

Example: return

public static int findFirstEven(int[] arr) {
    for (int num : arr) {
        if (num % 2 == 0) {
            return num; // returns the first even number found
        }
    }
    return -1; // returns -1 if no even number is found
}
