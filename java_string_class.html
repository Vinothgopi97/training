Java String Class

The String class in Java is used to represent a sequence of characters. Strings are widely used in Java programming for storing and manipulating text.

Key Points

  • String is a class in java.lang package.
  • Strings in Java are immutable (once created, their values cannot be changed).
  • String literals are stored in a special memory area called the String Constant Pool.
  • Commonly created using string literals or the new keyword.

Creating Strings

String s1 = "Hello";              // Using string literal
String s2 = new String("World");  // Using new keyword

Immutability of Strings

When you modify a string, a new object is created. The original string remains unchanged. 

String s = "Java";
s.concat(" Programming");
System.out.println(s); // Output: Java

String s2 = s.concat(" Programming");
System.out.println(s2); // Output: Java Programming

Common String Methods

  • length() - Returns the length of the string.
  • charAt(int index) - Returns the character at the specified index.
  • substring(int beginIndex, int endIndex) - Returns a substring.
  • toLowerCase(), toUpperCase() - Converts case.
  • trim() - Removes leading and trailing spaces.
  • equals(), equalsIgnoreCase() - Compares strings.
  • compareTo() - Lexicographically compares two strings.
  • replace() - Replaces characters or substrings.
  • split() - Splits the string based on a regex.

Example

public class StringExample {
    public static void main(String[] args) {
        String str = "  Hello Java!  ";
        System.out.println("Original: '" + str + "'");
        System.out.println("Trimmed: '" + str.trim() + "'");
        System.out.println("Uppercase: " + str.toUpperCase());
        System.out.println("Substring: " + str.substring(2, 7));
        System.out.println("Length: " + str.length());
        System.out.println("Char at 4: " + str.charAt(4));
        System.out.println("Replace: " + str.replace("Java", "World"));
        System.out.println("Equals 'Hello Java!': " + str.trim().equals("Hello Java!"));
    }
}

String Comparison

String a = "Java";
String b = "Java";
String c = new String("Java");

System.out.println(a == b);        // true (same reference in pool)
System.out.println(a == c);        // false (different objects)
System.out.println(a.equals(c));   // true (same value)

Why Strings are Immutable?

  • Security: Used in sensitive operations like file paths, network connections.
  • Thread safety: Immutable objects are inherently thread-safe.
  • String pool: Allows sharing of string literals for memory efficiency.
