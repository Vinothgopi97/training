Java 10 - Important Concepts

1. Local-Variable Type Inference (var)

Java 10 introduced the var keyword, allowing local variable types to be inferred by the compiler. This makes code more concise while retaining type safety. 


var list = List.of("a", "b", "c"); // inferred as List<String>
for (var item : list) {
    System.out.println(item);
}

2. Application Class-Data Sharing (AppCDS)

Application Class-Data Sharing allows class metadata to be shared between Java processes, reducing startup time and memory footprint. 


# Example usage (command line)
$ java -Xshare:dump -XX:SharedClassListFile=classes.lst -XX:SharedArchiveFile=app-cds.jsa -cp myapp.jar
$ java -XX:SharedArchiveFile=app-cds.jsa -cp myapp.jar Main

3. Other Enhancements

  • Optional.orElseThrow(): Optional.get() is deprecated; use orElseThrow() instead.
  • Unmodifiable Collections: copyOf() methods for List, Set, and Map create unmodifiable copies.

List<String> original = List.of("a", "b");
List<String> copy = List.copyOf(original); // unmodifiable copy
