Assignment: Java String Class & Exception Handling

  1. Write a Java program to compare two strings using both == and equals() and explain the output.
    Example: Compare "Java" and new String("Java").
  2. Write a Java program to demonstrate the concept of the String Constant Pool in Java.
    Example: Create two string literals with the same value and compare their references.
  3. Write a Java program to reverse a string without using the built-in reverse() method.
    Example: Input: "hello" → Output: "olleh"
  4. Write a Java program to show the output of the following code:
    String s1 = "hello";
String s2 = new String("hello");
System.out.println(s1 == s2);
System.out.println(s1.equals(s2));
    Example: Run the code and explain the output.
  5. Write a Java program to convert a string to uppercase and remove leading/trailing spaces.
    Example: Input: " java programming " → Output: "JAVA PROGRAMMING"
  6. Write a Java method to count the number of vowels in a given string.
    Example: Input: "Education" → Output: 5
  7. Write a Java program to demonstrate string immutability.
    Example: Show that concatenating to a string does not change the original string.
  8. Write a Java program to split a string into an array of words.
    Example: Input: "Java is fun" → Output: ["Java", "is", "fun"]
  9. Write a Java program to create a custom exception and show how to throw and catch it.
    Example: Throw InvalidAgeException if age is less than 18.
  10. Write a Java program to demonstrate the difference between throw and throws keywords.
    Example: Create a method that throws an exception using both keywords.
  11. Write a Java program to show that a try block can exist without a catch block (using finally).
    Example: Use try-finally to print a message.
  12. Write a Java program to demonstrate the use of the finally block and show that it executes even if there is a return statement in the try block.
    Example: Return from try and print a message in finally.
  13. Write a Java program to read a string from the user and throw a custom exception if the string is empty.
    Example: If input is "", throw EmptyStringException.
  14. Write a Java program to handle multiple exceptions in a single catch block (multi-catch).
    Example: Handle ArithmeticException and ArrayIndexOutOfBoundsException together.
  15. Write a Java program where an exception is thrown in a method but not caught there, and is handled in the calling method.
    Example: Method foo() throws an exception, main() catches it.
