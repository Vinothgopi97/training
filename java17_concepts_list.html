Java 17 - Important Concepts (LTS)

1. Sealed Classes (Standard)

Sealed classes became a standard feature, allowing class hierarchies to be more strictly controlled. 


public sealed interface Service permits EmailService, SmsService {}
final class EmailService implements Service {}
final class SmsService implements Service {}

2. Pattern Matching for switch (Preview)

Pattern matching for switch allows more concise and type-safe switch statements. 


static String formatter(Object o) {
    return switch (o) {
        case Integer i -> "int " + i;
        case Long l    -> "long " + l;
        case String s  -> "String " + s;
        default        -> o.toString();
    };
}

3. Strongly Encapsulate JDK Internals

Most internal elements of the JDK are now strongly encapsulated, improving security and maintainability.
