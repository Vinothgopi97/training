Java Operators Explained

Operators in Java are special symbols or keywords that are used to perform operations on operands. Java supports several types of operators:

1. Arithmetic Operators

OperatorDescriptionExample
+Additiona + b
-Subtractiona - b
*Multiplicationa * b
/Divisiona / b
%Modulus (Remainder)a % b
++Incrementa++
--Decrementa--
Real-time Example: 
// Calculate total price for items in a shopping cart
int quantity = 5;
int pricePerItem = 20;
int total = quantity * pricePerItem; // 100

// Find out how many dozens and remaining eggs
int eggs = 29;
int dozens = eggs / 12; // 2
int remaining = eggs % 12; // 5

2. Assignment Operators

OperatorDescriptionExample
=Assigns valuea = 5
+=Add and assigna += 3 // a = a + 3
-=Subtract and assigna -= 2 // a = a - 2
*=Multiply and assigna *= 4 // a = a * 4
/=Divide and assigna /= 2 // a = a / 2
%=Modulus and assigna %= 3 // a = a % 3
Real-time Example: 
// Add bonus points to a user's score
int score = 50;
score += 10; // score is now 60

// Apply a discount of 20% on price
double price = 200.0;
price *= 0.8; // price is now 160.0

3. Relational Operators

OperatorDescriptionExample
==Equal toa == b
!=Not equal toa != b
>Greater thana > b
<Less thana < b
>=Greater than or equal toa >= b
<=Less than or equal toa <= b
Real-time Example: 
// Check if user is eligible to vote
int age = 19;
if (age >= 18) {
    System.out.println("Eligible to vote");
}

// Compare two scores
int scoreA = 85, scoreB = 90;
if (scoreA != scoreB) {
    System.out.println("Scores are different");
}

4. Logical Operators

OperatorDescriptionExample
&&Logical ANDa > 5 && b < 10
||Logical ORa > 5 || b < 10
!Logical NOT!(a == b)
Real-time Example: 
// Check if a user can access a resource
boolean isAdmin = true;
boolean hasPermission = false;
if (isAdmin || hasPermission) {
    System.out.println("Access granted");
}

// Validate input
int age = 25;
if (age > 18 && age < 60) {
    System.out.println("Valid age for registration");
}

5. Bitwise Operators

OperatorDescriptionExample
&Bitwise ANDa & b
|Bitwise ORa | b
^Bitwise XORa ^ b
~Bitwise NOT~a
<<Left shifta << 2
>>Right shifta >> 2
>>>Unsigned right shifta >>> 2
Real-time Example: 
// Set and check permission flags using bitwise operators
int READ = 1;    // 0001
int WRITE = 2;   // 0010
int EXECUTE = 4; // 0100

int permissions = READ | WRITE; // 0011 (3)
boolean canWrite = (permissions & WRITE) != 0; // true

// Efficient multiplication/division by powers of 2
int n = 8;
int doubled = n << 1; // 16
int halved = n >> 1;  // 4

6. Conditional (Ternary) Operator

condition ? expr1 : expr2 â€” Returns expr1 if condition is true, else expr2.

Real-time Example: 
// Find the greater of two numbers
int a = 10, b = 20;
int max = (a > b) ? a : b; // max is 20

// Display message based on login status
boolean loggedIn = false;
String message = loggedIn ? "Welcome back!" : "Please log in.";

7. instanceof Operator

Checks if an object is an instance of a particular class or interface.

Real-time Example: 
Object obj = "Hello";
if (obj instanceof String) {
    System.out.println(((String)obj).toUpperCase()); // Prints "HELLO"
}

8. Miscellaneous Operators

  • new: Used to create new objects. MyClass obj = new MyClass();
  • Dot (.): Access members of a class or object. obj.method()
Real-time Example: 
// Creating a new ArrayList and adding elements
List names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
System.out.println(names.size()); // 2

Operator Precedence in Java

Operator precedence determines the order in which operators are evaluated in expressions. Operators with higher precedence are evaluated before operators with lower precedence. Use parentheses () to explicitly specify evaluation order.

Precedence Level (Highest to Lowest) Operators Description
1 (), [], . Parentheses, array subscript, member access
2 ++, --, ~, !, +, -, (type), new Unary operators, type cast, object creation
3 *, /, % Multiplicative
4 +, - Additive
5 <<, >>, >>> Shift
6 <, <=, >, >=, instanceof Relational
7 ==, != Equality
8 & Bitwise AND
9 ^ Bitwise XOR
10 | Bitwise OR
11 && Logical AND
12 || Logical OR
13 ?: Ternary (conditional)
14 =, +=, -=, *=, /=, %=, &=, ^=, |=, <<=, >>=, >>>= Assignment

Example:

// Calculate final bill with discount and tax
double bill = 1000;
double discount = 0.1;
double tax = 0.05;
double finalAmount = bill - bill * discount + bill * tax;
// * and / are evaluated before + and -

For more details, see the Java documentation on operators.
