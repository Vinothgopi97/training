Java 11 - Important Concepts

1. New String Methods

Java 11 introduced several new methods for the String class, such as isBlank(), lines(), strip(), stripLeading(), stripTrailing(), and repeat(int). 


String s = "  Hello World  ";
System.out.println(s.isBlank()); // false
System.out.println("   ".isBlank()); // true
System.out.println(s.strip()); // "Hello World"
System.out.println("Line1\nLine2".lines().count()); // 2
System.out.println("abc".repeat(3)); // abcabcabc

2. Local-Variable Syntax for Lambda Parameters

You can use var in lambda parameters for local variable type inference. 


BiFunction add = (var x, var y) -> x + y;

3. HTTP Client API

The new java.net.http.HttpClient API provides a modern way to perform HTTP requests. 


HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://example.com"))
    .build();
HttpResponse response = client.send(request, HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());

4. Removal of Java EE and CORBA Modules

Java EE modules (like JAXB, JAX-WS) and CORBA modules were removed from the JDK. You need to add them as external dependencies if required.
