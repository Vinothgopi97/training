Map Interface in Java

The Map interface represents a collection of key-value pairs. Each key is unique and maps to exactly one value.

  • Common implementations: HashMap, LinkedHashMap, TreeMap
  • Keys are unique, values can be duplicated
  • Allows one null key (HashMap, LinkedHashMap)

Example

import java.util.*;

Map<String, Integer> map = new HashMap<>();
map.put("A", 1);
map.put("B", 2);
map.put("A", 3); // Overwrites value for key "A"
System.out.println(map); // {A=3, B=2}
System.out.println(map.get("B")); // 2

Tip: Use Map for fast lookups by key.

Common Map Methods

  • put(K key, V value): Associates value with key.
  • get(Object key): Returns value for key, or null if not found.
  • remove(Object key): Removes mapping for key.
  • containsKey(Object key): Checks if key exists.
  • containsValue(Object value): Checks if value exists.
  • size(): Number of key-value pairs.
  • clear(): Removes all mappings.
  • isEmpty(): Checks if map is empty.
  • keySet(): Returns a Set view of the keys.
  • values(): Returns a Collection view of the values.
  • entrySet(): Returns a Set view of the mappings.
