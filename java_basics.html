Java Basics for Beginners

What is Java?

Java is a high-level, object-oriented programming language. It is platform-independent, meaning Java code can run on any device that has a Java Virtual Machine (JVM).

Java Keywords

Keywords are reserved words in Java that have a special meaning and cannot be used as identifiers (names for variables, classes, etc.). Some important Java keywords:

  • class: Declares a class.
  • public: Access modifier, visible to all.
  • static: Belongs to the class, not an instance.
  • void: No return value.
  • int, double, boolean: Data types.
  • if, else, for, while: Control flow.
  • return: Exits a method and optionally returns a value.
  • new: Creates new objects.
  • this: Refers to the current object.

Basic Java Program Structure

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Memory Management in Java

  • Stack Memory: Stores local variables and function calls. Managed automatically.
  • Heap Memory: Stores objects created with new. Managed by the Garbage Collector.
  • Garbage Collection: Java automatically frees memory used by objects that are no longer referenced.

Example: 

String name = new String("Java"); // "name" is a reference on the stack, the String object is on the heap.

Variables and Data Types

  • byte: 8-bit integer. Range: -128 to 127
  • short: 16-bit integer. Range: -32,768 to 32,767
  • int: 32-bit integer numbers (e.g., 5, -3). Range: -2,147,483,648 to 2,147,483,647
  • long: 64-bit integer. Range: -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
  • float: 32-bit floating-point decimal numbers. Range: approximately ±1.4e-45 to ±3.4e+38
  • double: 64-bit floating-point decimal numbers (e.g., 3.14). Range: approximately ±4.9e-324 to ±1.8e+308
  • boolean: true or false. Range: Only true or false
  • char: Single character (e.g., 'A'). Range: 0 to 65,535 (16-bit Unicode character)
  • String: Sequence of characters (e.g., "Hello"). Range: Limited by memory (no fixed size)

Why is String a Class and Not a Primitive Data Type?

In Java, String is a class, not a primitive data type, because strings are complex objects that can store sequences of characters and provide many useful methods (like length(), substring(), equals(), etc.). Primitive data types (such as int, double, char, boolean) are simple and store only single values, while String objects can represent text of any length and support various operations.

How are Strings Stored in Java?

Java uses a special memory area called the String pool to store string literals. When you create a string literal (e.g., String s = "Hello";), Java checks if the value already exists in the pool. If it does, the reference is reused; if not, a new string object is created in the pool. This improves memory efficiency and performance for string handling.

Example: 

String a = "Java"; // Stored in the String pool
String b = "Java"; // Reuses the same object from the pool
String c = new String("Java"); // Creates a new object in the heap, not in the pool

What are Primitive Data Types?

Primitive data types in Java are the most basic data types built into the language. They are not objects and store simple values directly in memory. Java has eight primitive data types: byte, short, int, long, float, double, char, and boolean. These types are efficient and fast, and they do not provide methods or additional functionality like classes do.

Control Flow

if (x > 0) {
    System.out.println("Positive");
} else {
    System.out.println("Non-positive");
}

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Summary

  • Java is object-oriented and platform-independent.
  • Keywords have special meanings and cannot be used as names.
  • Memory is managed automatically, with garbage collection for unused objects.
  • Basic program structure includes classes and the main method.
