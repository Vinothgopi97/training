Java Keywords Explained

Java keywords are reserved words that have a predefined meaning in the language. They cannot be used as identifiers (names for variables, classes, etc.). Here are the main types of keywords in Java, grouped by their purpose:

1. Access Modifiers

  • public: Accessible from any other class.
  • private: Accessible only within the declared class.
  • protected: Accessible within the same package and subclasses.

2. Class, Method, and Variable Related

  • class: Declares a class.
  • interface: Declares an interface.
  • extends: Indicates inheritance from a superclass.
  • implements: Indicates implementation of an interface.
  • abstract: Declares an abstract class or method.
  • final: Prevents inheritance or modification.
  • static: Belongs to the class, not an instance.
  • void: Indicates no return value.
  • new: Creates new objects.
  • this: Refers to the current object.
  • super: Refers to the superclass object.

3. Data Types

  • byte, short, int, long: Integer types of various sizes.
  • float, double: Floating-point numbers.
  • char: Single 16-bit Unicode character.
  • boolean: true or false values.

4. Control Flow

  • if, else: Conditional statements.
  • switch, case, default: Multi-way branch statement.
  • for, while, do: Looping statements.
  • break, continue: Control loop execution.
  • return: Exits a method and optionally returns a value.

5. Exception Handling

  • try, catch, finally: Handle exceptions.
  • throw: Throws an exception.
  • throws: Declares exceptions a method can throw.

6. Package and Import

  • package: Defines a package.
  • import: Imports other Java packages or classes.

7. Others

  • instanceof: Tests if an object is an instance of a class.
  • enum: Defines an enumerated type.
  • synchronized: Used for thread safety.
  • transient: Prevents serialization.
  • volatile: Indicates a variable may be changed by multiple threads.
  • assert: Used for debugging.
  • native: Specifies that a method is implemented in native code (outside Java).
  • strictfp: Restricts floating-point calculations to ensure portability.

Reserved but Unused Keywords

  • const, goto: Reserved for future use, not used in Java.
