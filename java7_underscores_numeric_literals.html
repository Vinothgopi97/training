Java 7: Underscores in Numeric Literals

Java 7 allows the use of underscores in numeric literals to improve readability of large numbers.

Example

public class UnderscoreNumericLiteralsExample {
    public static void main(String[] args) {
        int million = 1_000_000;
        double pi = 3.14_15_92;
        System.out.println("Million: " + million);
        System.out.println("Pi: " + pi);
    }
}
  • Underscores can be placed between digits in numeric literals.
  • They cannot be at the start or end of a number, or next to a decimal point.
