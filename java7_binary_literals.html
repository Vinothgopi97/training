Java 7: Binary Literals

Java 7 introduced binary literals, allowing you to represent integer values in binary format using the 0b or 0B prefix.

Example

public class BinaryLiteralsExample {
    public static void main(String[] args) {
        int binary = 0b1010; // 10 in decimal
        System.out.println("Binary 0b1010 = " + binary);
    }
}
  • Binary literals can be used with byte, short, int, and long types.
