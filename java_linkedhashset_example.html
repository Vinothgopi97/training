LinkedHashSet in Java

LinkedHashSet is a Set implementation that maintains insertion order using a linked list.

  • Maintains insertion order
  • Allows one null value
  • Not synchronized

Example

import java.util.*;

LinkedHashSet<Integer> set = new LinkedHashSet<>();
set.add(3);
set.add(1);
set.add(2);
System.out.println(set); // [3, 1, 2]
set.remove(1);
System.out.println(set); // [3, 2]

Tip: Use LinkedHashSet when you need uniqueness and order.

Common LinkedHashSet Methods

  • add(E e): Adds element if not present.
  • remove(Object o): Removes specified element.
  • contains(Object o): Checks if element exists.
  • size(): Number of elements.
  • clear(): Removes all elements.
  • isEmpty(): Checks if set is empty.
  • iterator(): Returns an iterator over the elements.
