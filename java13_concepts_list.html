Java 13 - Important Concepts

1. Text Blocks (Preview)

Text blocks provide a new way to declare multi-line string literals using """. 


String html = """
    <html>
      <body>
        Hello, world!
      </body>
    </html>
    """;
System.out.println(html);

2. Switch Expressions (2nd Preview)

Further refinements to switch expressions, allowing both statement and expression forms. 


String dayType = switch (day) {
    case 1, 7 -> "Weekend";
    default -> "Weekday";
};
