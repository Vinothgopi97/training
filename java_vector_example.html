Vector in Java

Vector is a synchronized, growable array of objects. It is thread-safe but slower than ArrayList.

  • Synchronized (thread-safe)
  • Allows duplicates and nulls
  • Legacy class, rarely used in new code

Example

import java.util.*;

Vector<Integer> numbers = new Vector<>();
numbers.add(1);
numbers.add(2);
numbers.add(3);
numbers.remove(1); // removes element at index 1
System.out.println(numbers); // [1, 3]

Common Vector Methods

  • add(E e): Adds element to end.
  • addElement(E obj): Adds element (legacy method).
  • remove(int index): Removes element at index.
  • removeElement(Object obj): Removes first occurrence (legacy).
  • elementAt(int index): Gets element at index.
  • firstElement(): Gets first element.
  • lastElement(): Gets last element.
  • capacity(): Returns current capacity.
  • size(): Number of elements.
  • isEmpty(): Checks if empty.

Tip: Prefer ArrayList for single-threaded code, use Vector only if thread safety is required and you cannot use concurrent collections.
