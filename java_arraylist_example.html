ArrayList in Java

ArrayList is a resizable array implementation of the List interface. It allows random access and is best for storing and accessing data.

  • Allows duplicates and nulls
  • Maintains insertion order
  • Not synchronized (not thread-safe)
  • Fast random access, slow insert/remove in the middle

Example

import java.util.*;

ArrayList<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Orange");
fruits.remove("Banana");
System.out.println(fruits); // [Apple, Orange]
System.out.println(fruits.get(1)); // Orange

Tip: Use ArrayList when you need fast access by index and don't need thread safety.

Common ArrayList Methods

  • add(E e): Adds an element to the end.
  • addAll(Collection<? extends E> c): Adds all elements from another collection.
  • get(int index): Gets element at index.
  • set(int index, E element): Sets element at index.
  • remove(int index): Removes element at index.
  • remove(Object o): Removes first occurrence of object.
  • size(): Number of elements.
  • clear(): Removes all elements.
  • contains(Object o): Checks if element exists.
  • ensureCapacity(int minCapacity): Increases capacity if needed.
  • trimToSize(): Trims capacity to current size.
