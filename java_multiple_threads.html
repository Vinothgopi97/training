Creating Multiple Threads in Java

You can create and start multiple threads in Java by creating multiple Thread objects and calling start() on each.

Example: Multiple Threads

class Worker extends Thread {
    private String name;
    Worker(String name) { this.name = name; }
    public void run() {
        for (int i = 1; i <= 3; i++) {
            System.out.println(name + " - Count: " + i);
        }
    }
    public static void main(String[] args) {
        Worker t1 = new Worker("Thread-1");
        Worker t2 = new Worker("Thread-2");
        Worker t3 = new Worker("Thread-3");
        t1.start();
        t2.start();
        t3.start();
    }
}

Output

The output order may vary because threads run concurrently.
