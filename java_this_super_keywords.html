this and super Keywords in Java

this Keyword

this is a reference variable in Java that refers to the current object (the object whose method or constructor is being called).

  • Used to refer to the current object's fields and methods.
  • Commonly used to resolve ambiguity between instance variables and parameters with the same name.
Example: 
class Student {
    String name;
    Student(String name) {
        this.name = name; // 'this.name' refers to the instance variable
    }
}
Usage in method call: 
class Demo {
    void show() {
        System.out.println("Hello");
    }
    void display() {
        this.show(); // calls show() of current object
    }
}

super Keyword

super is a reference variable in Java used to refer to the immediate parent class object.

  • Used to access parent class fields, methods, and constructors.
  • Commonly used when a subclass overrides methods or fields of the parent class.
Example: Accessing parent class variable 
class Animal {
    String name = "Animal";
}
class Dog extends Animal {
    String name = "Dog";
    void printNames() {
        System.out.println(name);        // prints "Dog"
        System.out.println(super.name);  // prints "Animal"
    }
}
Example: Calling parent class method 
class Animal {
    void sound() {
        System.out.println("Animal sound");
    }
}
class Dog extends Animal {
    void sound() {
        System.out.println("Dog barks");
    }
    void printSound() {
        super.sound(); // calls Animal's sound()
    }
}
Example: Calling parent class constructor 
class Animal {
    Animal() {
        System.out.println("Animal constructor");
    }
}
class Dog extends Animal {
    Dog() {
        super(); // calls Animal's constructor
        System.out.println("Dog constructor");
    }
}
