Java Collections Assignment Questions

  13. Question 13: Write a Java program to remove duplicate elements from a List<String> using a Set.
  14. Question 14: Write a Java program to iterate over a Map<String, Integer> and print each key-value pair.
  15. Question 15: Write a Java program to check if a List<Integer> contains a specific value.
  16. Question 16: Write a Java program to convert a Set<String> to a List<String>.
  17. Question 17: Write a Java program to create a Stack<Integer>, push five elements, and pop all elements displaying them.
  18. Question 18: Write a Java program to create a Queue<String> using LinkedList, add elements, and remove them one by one.
  19. Question 19: Write a Java program to find the intersection of two Set<Integer> objects.
  20. Question 20: Write a Java program to sort a Map<String, Integer> by its values.
