Java Programming Assignments

  1. Write a Java program to print "Hello, World!" to the console.
  2. Declare variables of all primitive data types in Java and print their default values.
  3. Write a Java program to swap two numbers using a temporary variable.
  4. Write a Java program to calculate the sum, difference, product, and quotient of two numbers entered by the user.
    Hint: Use Scanner to take input. Use +, -, *, and / operators for calculations.
  5. Write a Java program to check if a number is even or odd using an if-else statement.
    Hint: Use the modulus operator % to check divisibility by 2.
  6. Write a Java program to find the largest of three numbers using nested if-else statements.
    Hint: Compare the numbers pairwise using if-else blocks.
  7. Write a Java program to print the multiplication table of a given number using a for loop.
    Hint: Loop from 1 to 10 and multiply the given number by the loop variable in each iteration.
  8. Write a Java program to calculate the factorial of a number using a while loop.
    Hint: Factorial of n is the product of all positive integers up to n (i.e., n! = n × (n-1) × ... × 1). Use a loop to multiply numbers from 1 to n.
  9. Write a Java program to print all even numbers from 1 to 100 using a for loop.
    Hint: Loop from 1 to 100 and print the number if it is divisible by 2.
  10. Write a Java program to reverse a given integer number using a while loop.
    Hint: Use modulus % to get the last digit and integer division / to remove the last digit. Build the reversed number digit by digit.
  11. Write a Java program to check whether a character is a vowel or consonant using a switch statement.
    Hint: Use switch to compare the character against 'a', 'e', 'i', 'o', 'u' (and uppercase variants).
  12. Write a Java program to calculate the sum of digits of a given number.
    Hint: Use a loop to extract each digit using % 10 and add it to a sum variable. Remove the last digit using / 10.
  13. Write a Java program to print the Fibonacci series up to n terms using a for loop.
    Hint: The Fibonacci series starts with 0 and 1. Each next term is the sum of the previous two terms. Use a loop to generate the series.
  14. Write a Java program to find the GCD (Greatest Common Divisor) of two numbers.
    Hint: Use the Euclidean algorithm: repeatedly replace the larger number by the remainder of dividing the larger by the smaller, until one becomes zero. The other is the GCD.
  15. Write a Java program to count the number of positive, negative, and zero values entered by the user (use a loop and if-else).
    Hint: Use counters for each type and increment them based on the value entered in each iteration.
