Stack in Java

Stack is a subclass of Vector that implements a standard last-in, first-out (LIFO) stack.

  • Legacy class, prefer Deque for new code
  • Methods: push, pop, peek, empty

Common Stack Methods

  • push(E item): Pushes item onto stack.
  • pop(): Removes and returns top item.
  • peek(): Returns top item without removing.
  • empty(): Checks if stack is empty.
  • search(Object o): Returns 1-based position from top, or -1 if not found.

Example

import java.util.*;

Stack<String> stack = new Stack<>();
stack.push("A");
stack.push("B");
System.out.println(stack.peek()); // B
System.out.println(stack.pop());  // B
System.out.println(stack);        // [A]

Tip: For stack operations, prefer ArrayDeque for better performance.
