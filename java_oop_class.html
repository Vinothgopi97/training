Java OOP: Class

A class in Java is a blueprint for creating objects. It defines properties (fields) and behaviors (methods) that the objects created from the class will have. 

public class Car {
    // Fields (properties)
    String brand;
    int year;

    // Constructor
    public Car(String brand, int year) {
        this.brand = brand;
        this.year = year;
    }

    // Method (behavior)
    public void displayInfo() {
        System.out.println("Brand: " + brand + ", Year: " + year);
    }
}

Here, Car is a class with fields brand and year, a constructor, and a method displayInfo().
