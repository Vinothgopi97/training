Java Threads - Simple Assignment Questions

  1. Create a thread by extending the Thread class and print "Hello from thread!" five times.
    Example:
    class MyThread extends Thread {
      public void run() {
        for(int i=0;i<5;i++) System.out.println("Hello from thread!");
      }
    }
  2. Create a thread using the Runnable interface and print numbers from 1 to 5.
    Example:
    class MyRunnable implements Runnable {
      public void run() {
        for(int i=1;i<=5;i++) System.out.println(i);
      }
    }
  3. Start two threads: one prints "A" and the other prints "B" ten times each.
    Example:
    // Thread 1 prints "A", Thread 2 prints "B"
  4. Write a program where the main thread waits for another thread to finish using join().
    Example:
    t.join(); // main waits for t to finish
  5. Demonstrate thread sleep: make a thread print "Tick" every second for 5 seconds.
    Example:
    Thread.sleep(1000); // sleep for 1 second
  6. Show how to interrupt a thread and handle InterruptedException.
    Example:
    if(Thread.interrupted()) break;
  7. Create a shared counter variable and increment it from two threads. Use synchronized to avoid errors.
    Example:
    synchronized void increment() { counter++; }
  8. Demonstrate thread priority by setting different priorities for two threads.
    Example:
    t1.setPriority(Thread.MAX_PRIORITY);
  9. Print the current thread's name using Thread.currentThread().getName().
    Example:
    System.out.println(Thread.currentThread().getName());
