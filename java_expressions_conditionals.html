Java Expressions & Conditional Flow Statements

1. Expressions in Java

An expression in Java is a combination of variables, values, operators, and method calls that evaluates to a single value.

  • Arithmetic Expression: int sum = a + b;
  • Boolean Expression: boolean isAdult = age >= 18;
  • Assignment Expression: x = 5;

Example:

int a = 10;
int b = 5;
int sum = a + b; // sum is 15
boolean isGreater = sum > 10; // isGreater is true

2. Conditional Flow Statements

Conditional flow statements allow you to control the flow of your program based on conditions.

2.1 if Statement

Executes a block of code if a condition is true. 

int age = 20;
if (age >= 18) {
    System.out.println("You are an adult.");
}

2.2 if-else Statement

Executes one block if the condition is true, another if it is false. 

int number = 7;
if (number % 2 == 0) {
    System.out.println("Even number");
} else {
    System.out.println("Odd number");
}

2.3 else-if Ladder

Checks multiple conditions in sequence. 

int marks = 85;
if (marks >= 90) {
    System.out.println("Grade A");
} else if (marks >= 75) {
    System.out.println("Grade B");
} else {
    System.out.println("Grade C");
}

2.4 switch Statement

Selects one of many code blocks to execute.

  • With break: Each case ends with break to prevent fall-through.
  • Without break: Execution continues ("falls through") to the next case until a break or end of switch.
  • With default: default runs if no case matches.
  • Without default: If no case matches and no default, nothing runs.

Examples:

// With break (recommended)
int day = 3;
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Other day");
}

// Without break (fall-through)
int num = 2;
switch (num) {
    case 1:
        System.out.println("One");
    case 2:
        System.out.println("Two");
    case 3:
        System.out.println("Three");
    default:
        System.out.println("Other number");
}
// Output when num = 2:
// Two
// Three
// Other number

// Without default
int color = 4;
switch (color) {
    case 1:
        System.out.println("Red");
        break;
    case 2:
        System.out.println("Green");
        break;
    case 3:
        System.out.println("Blue");
        break;
}
// No output since color is 4 and no default is present

// With default only
int option = 10;
switch (option) {
    default:
        System.out.println("No valid option selected");
}

Summary

  • Expressions compute values.
  • Conditional statements control program flow based on conditions.
