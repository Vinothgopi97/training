Java OOP: Object

An object is an instance of a class. It has its own state and can use the methods defined in the class. 

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car("Toyota", 2020);
        myCar.displayInfo(); // Output: Brand: Toyota, Year: 2020
    }
}

Here, myCar is an object of the Car class, created using the constructor and using its method.
