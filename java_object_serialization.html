Storing and Retrieving Objects as Files in Java (Serialization & Deserialization)

In Java, serialization is the process of converting an object into a byte stream so it can be saved to a file or transmitted over a network. Deserialization is the reverse processâ€”reconstructing the object from the byte stream.

1. Making a Class Serializable

To serialize an object, the class must implement the java.io.Serializable interface. 


import java.io.Serializable;

public class Person implements Serializable {
    private String name;
    private int age;
    // Constructor, getters, setters
}

2. Writing an Object to a File (Serialization)


import java.io.FileOutputStream;
import java.io.ObjectOutputStream;

Person person = new Person("Alice", 30);
try (ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("person.ser"))) {
    oos.writeObject(person);
    System.out.println("Object serialized to person.ser");
} catch (Exception e) {
    e.printStackTrace();
}

3. Reading an Object from a File (Deserialization)


import java.io.FileInputStream;
import java.io.ObjectInputStream;

try (ObjectInputStream ois = new ObjectInputStream(new FileInputStream("person.ser"))) {
    Person p = (Person) ois.readObject();
    System.out.println("Name: " + p.getName() + ", Age: " + p.getAge());
} catch (Exception e) {
    e.printStackTrace();
}

4. Notes

  • All fields of the class (and referenced objects) must be serializable.
  • Mark fields as transient if you do not want them to be serialized.
  • Serialization is commonly used for saving application state, caching, or sending objects over a network.
