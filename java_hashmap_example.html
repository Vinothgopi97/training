HashMap in Java

HashMap is an implementation of the Map interface that stores key-value pairs with no guaranteed order.

  • Allows one null key and multiple null values
  • Not synchronized
  • Fast put, get, remove operations

Example

import java.util.*;

HashMap<String, Integer> map = new HashMap<>();
map.put("One", 1);
map.put("Two", 2);
map.put(null, 3);
System.out.println(map); // {null=3, One=1, Two=2}
map.remove("Two");
System.out.println(map); // {null=3, One=1}

Common HashMap Methods

  • put(K key, V value): Adds or updates key-value pair.
  • get(Object key): Gets value for key.
  • remove(Object key): Removes mapping for key.
  • containsKey(Object key): Checks if key exists.
  • containsValue(Object value): Checks if value exists.
  • size(): Number of entries.
  • clear(): Removes all entries.
  • isEmpty(): Checks if map is empty.
  • keySet(): Set of keys.
  • values(): Collection of values.
  • entrySet(): Set of key-value pairs.

Tip: Use HashMap for fast, unordered key-value storage.
