Logger Classes in Java

Java provides built-in logging capabilities via the java.util.logging package. The main class is Logger, which is used to log messages at different levels (INFO, WARNING, SEVERE, etc.).

Basic Usage Example


import java.util.logging.Logger;

public class LoggingExample {
    private static final Logger logger = Logger.getLogger(LoggingExample.class.getName());

    public static void main(String[] args) {
        logger.info("This is an info message");
        logger.warning("This is a warning message");
        logger.severe("This is a severe message");
    }
}

Customizing Logger

You can configure log levels, handlers, and formatters for more control. 


logger.setLevel(Level.FINE);
logger.fine("This is a fine-grained debug message");
