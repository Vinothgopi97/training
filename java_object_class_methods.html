Object Class and Its Methods in Java

The Object class is the root class of the Java class hierarchy. Every class in Java implicitly extends the Object class if no other superclass is specified.

Commonly Used Methods of Object Class

  • toString() — Returns a string representation of the object.
  • equals(Object obj) — Compares this object to another for equality.
  • hashCode() — Returns a hash code value for the object.
  • getClass() — Returns the runtime class of the object.
  • clone() — Creates and returns a copy of the object (requires implementing Cloneable).
  • finalize() — Called by the garbage collector before object is destroyed (rarely used).
  • wait(), notify(), notifyAll() — Used for thread synchronization.

Examples

toString()

class Student {
    String name;
    Student(String name) { this.name = name; }
    public String toString() {
        return "Student name: " + name;
    }
}
Student s = new Student("John");
System.out.println(s); // Output: Student name: John

equals() and hashCode()

class Student {
    String name;
    Student(String name) { this.name = name; }
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;
        Student s = (Student) obj;
        return name.equals(s.name);
    }
    public int hashCode() {
        return name.hashCode();
    }
}
Student s1 = new Student("John");
Student s2 = new Student("John");
System.out.println(s1.equals(s2)); // Output: true

getClass()

Student s = new Student("John");
System.out.println(s.getClass().getName()); // Output: Student

clone()

class Student implements Cloneable {
    String name;
    Student(String name) { this.name = name; }
    public Object clone() throws CloneNotSupportedException {
        return super.clone();
    }
}
Student s1 = new Student("John");
Student s2 = (Student) s1.clone();

Summary

  • All Java classes inherit methods from Object class.
  • Overriding toString(), equals(), and hashCode() is common for custom classes.
