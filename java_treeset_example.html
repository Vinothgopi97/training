TreeSet in Java

TreeSet is a NavigableSet implementation based on a TreeMap. It stores elements in sorted (natural or custom) order.

  • Does not allow null elements
  • Implements SortedSet and NavigableSet
  • Logarithmic time for basic operations

Example

import java.util.*;

TreeSet<String> set = new TreeSet<>();
set.add("Banana");
set.add("Apple");
set.add("Mango");
System.out.println(set); // [Apple, Banana, Mango]
System.out.println(set.first()); // Apple
System.out.println(set.last());  // Mango

Common TreeSet Methods

  • add(E e): Adds element if not present.
  • remove(Object o): Removes specified element.
  • contains(Object o): Checks if element exists.
  • first(): Returns the lowest element.
  • last(): Returns the highest element.
  • higher(E e): Returns least element greater than given element.
  • lower(E e): Returns greatest element less than given element.
  • size(): Number of elements.
  • clear(): Removes all elements.
  • isEmpty(): Checks if set is empty.
  • iterator(): Returns an iterator over the elements.

Tip: Use TreeSet when you need sorted unique elements.
