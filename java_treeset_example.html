TreeSet Example

TreeSet stores elements in a sorted (natural order) and unique way. 

import java.util.TreeSet;

TreeSet<String> set = new TreeSet<>();
set.add("Banana");
set.add("Apple");
set.add("Mango");
System.out.println(set); // [Apple, Banana, Mango]
  • Does not allow null elements
  • Implements SortedSet
