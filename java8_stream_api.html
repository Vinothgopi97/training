Stream API in Java 8

The Stream API allows processing collections of objects in a functional style. It supports operations like filter, map, reduce, and more.

Example:


List<String> names = Arrays.asList("A", "B", "C");
names.stream().filter(s -> s.startsWith("A")).forEach(System.out::println);

Streams can be sequential or parallel, and help write concise and readable code for data manipulation.
