Java 7: Diamond Operator (<>)

The diamond operator (<>) was introduced in Java 7 to simplify the use of generics. It allows the compiler to infer the type parameters from the context, reducing code verbosity.

Before Java 7

List<String> list = new ArrayList<String>();

With Diamond Operator (Java 7+)

List<String> list = new ArrayList<>();
  • The type on the right side is inferred from the left side.
  • Makes code cleaner and easier to read.

Example

import java.util.*;

public class DiamondOperatorExample {
    public static void main(String[] args) {
        Map<String, Integer> map = new HashMap<>();
        map.put("A", 1);
        map.put("B", 2);
        System.out.println(map);
    }
}
