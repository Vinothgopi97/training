Assignment: Java 7 Concepts

  1. Try-with-resources:
    Write a Java program to read a text file using BufferedReader and print its contents using try-with-resources. Ensure the file is closed automatically.
  2. Strings in switch:
    Write a Java program that takes a day name as a string and uses a switch statement to print whether it is a weekday or weekend.
  3. Binary literals:
    Write a Java program to declare integer variables using binary literals and print their decimal values.
  4. Underscores in numeric literals:
    Write a Java program to declare large numbers using underscores in numeric literals and print their values.
  5. Multi-catch block:
    Write a Java program that can throw either an ArithmeticException or ArrayIndexOutOfBoundsException and handle both using a single multi-catch block.
  6. Diamond operator:
    Write a Java program to create a List<String> using the diamond operator and add some elements to it. Print the list.
  7. Improved type inference for generic instance creation:
    Write a Java method that returns an empty ArrayList<Integer> using the diamond operator and type inference. Demonstrate calling this method.
  8. NIO.2 Filesystem API:
    Write a Java program using java.nio.file to create a file, write some text to it, read the text back, and then delete the file.
